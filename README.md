
# ToDo-App

Repositorio em desenvolvimento

## Rodando localmente

Clone o projeto

```bash
  git clone https://link-para-o-projeto
```

Entre no diretório do projeto

```bash
  cd my-project
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  npm run start
```


## Stack utilizada

**Front-end:** React, TypeScript, TailwindCSS, PostCSS

**Back-end:** Node, Express, TypeScript
